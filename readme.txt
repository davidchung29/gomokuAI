Gomoku AI


Running the Program


To watch two AI programs play :

Run /term project/aiTester.py

To play single player or against another person:

Run /term project/gomoku.py

In the console type '1' to play 1 player mode against an AI bot or type '2' to play 2 player mode


Libraries


The CS15-112 library is included in the file


Shortcuts:


When watching two AI play each other, pressing 'p' will pause the game and 'r' will restart the game

When playing 2 player or 1 player mode, pressing 'r' will restart the game, and pressing the up or down arrows will clear and resize the board.
